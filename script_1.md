# 1. 기본 문법 | 어순, 평서문, 의문문, 명령문, 의문사, 조동사

## 어순

어순: 언어마다 다르게 하고, 기본으로 고른 native 단어를 가지고 순서만 익히게 퀴즈 형식으로 냄.
예시: 한국어 native <-> 영어 learner 라고 하면 => 나는 원해 커피를. 이런 식으로 순서 퀴즈로 냄.
이 경우 가짓수가 많이 나오기 때문에 복잡하지 않게 가장 나은 DB 가져오기 방식을 택해야 함.
