# Oriverse 사용자 관점 피드백 (final-plan.md 기준)

> 작성일: 2026-03-05
> 관점: 앱을 실제로 사용하는 20-30대 초급 언어 학습자

---

## 1. 온보딩 불편함

### 1-1. 레벨 퀴즈 3문항은 너무 적다
- 3문항으로 초급/중급을 나누는 건 신뢰가 안 감
- 결과가 틀리면 처음부터 너무 쉽거나 너무 어려운 스크립트부터 시작하게 됨
- 퀴즈 결과를 나중에 직접 수정하는 방법이 명시되어 있지 않음

### 1-2. 세계 지도에서 언어 선택하는 방식이 헷갈릴 수 있다
- 스페인어를 배우고 싶은데 스페인을 탭해야 하는지, 멕시코를 탭해야 하는지 불명확
- 중국어는 대만/중국 어느 쪽?
- 지도 크기상 작은 국가(스위스 등 독일어권)는 누르기 어렵고, 같은 언어를 쓰는 나라가 여럿이면 혼란스러움

---

## 2. 학습 탭 불편함

### 2-1. 소통 탭 잠금 해제가 너무 늦다 (핵심 문제)
- 앱의 핵심 기능인 소통을 경험하기 위해 스텝 1~6 전체를 완료해야 함
- 처음 앱을 받아서 "소통 탭"을 열었는데 잠겨 있으면 → 앱을 지울 가능성 높음
- 스텝 6개를 끝내기 전에 "이게 재미있는 앱인지"를 체험할 방법이 없음

### 2-2. 학습 스텝 순서가 고정되어 있다
- 카페/음식점/마트 순서인데, 내가 지금 당장 필요한 건 "취미 대화"일 수도 있음
- 스텝을 건너뛰거나 원하는 스텝만 먼저 할 수 없음

### 2-3. AI 피드백이 텍스트 기반이라 발음 교정이 어렵다
- Whisper가 텍스트로 변환 후 예상 텍스트와 비교하는 방식
- 발음이 어색해도 텍스트 매칭이 되면 통과될 수 있음
- "발음은 어떻게 고쳐야 하지?"에 대한 답을 받기 어려움

### 2-4. 스텝 7 (50동사) 페이월 경험이 불편할 수 있다
- 처음 3개를 무료로 보여주다가 4번째에서 막히는 방식
- 유료로 전환하지 않으면 거기서 학습이 끊김
- 3개가 너무 적어서 가치 판단이 어려울 수 있음

## 3. 소통 탭 불편함

### 3-1. 무료 사용 제한이 두 겹으로 걸려 있다
- 6스텝 완료 조건 + 브로드캐스트 3회/일 + 1:1 대화 1인당 10회 제한
- 처음 소통 탭에 들어간 날부터 "이미 3번 썼어요"가 뜰 수 있음
- 제한이 많아서 앱이 답답하게 느껴질 수 있음

### 3-2. 브로드캐스트 하루 3회 제한이 너무 타이트하다
- 언어 학습 앱에서 적극적으로 연습하고 싶은 사용자에게는 3회가 금방 소진됨
- 제한에 걸리는 순간 당일 앱을 더 쓸 이유가 없어짐
- "일일 브로드캐스트 패스 $1.99"는 하루치치고 비싸게 느껴질 수 있음

### 3-3. 1:1 대화 10회 제한 후 관계가 끊기는 느낌
- 좋은 언어 파트너를 찾아서 대화를 이어가다가 갑자기 "10회 다 썼습니다"
- 추가 대화는 구독을 해야 하는데, 그 상대방을 위해 구독할 가치가 있는지 모름
- 10회가 얼마나 빨리 소진되는지 처음엔 감이 없음

### 3-4. 보이스메일 60초 제한
- 자연스러운 대화에서 60초 이상 말하는 경우가 있음
- 말이 잘리면 다시 녹음해야 하는데, 몇 번이나 해야 하는지 불명확

### 3-5. 모더레이션으로 인한 딜레이
- 브로드캐스트를 누른 후 바로 피드에 올라가지 않을 수 있음
- "내 메시지가 왜 안 보이지?"라는 혼란
- 거부됐을 때 이유 설명이 없으면 억울할 수 있음

### 3-6. 브로드캐스트 피드의 Cold Start 문제
- 초기에 사용자가 적으면 피드에 아무것도 없을 수 있음
- 6스텝을 어렵게 완료했는데 소통 탭에 아무 메시지도 없으면 허탈함
- 언어별로 사용자가 분산되면 특정 언어는 피드가 거의 비어 있을 수 있음

### 3-7. 상대방 정보가 너무 부족하다
- 브로드캐스트 피드에서 "이 사람이 어느 수준인지", "활동적인지" 알 수 없음
- 답장했는데 상대가 이미 탈퇴했거나 오래된 계정일 수 있음
- 상대방 프로필 페이지가 있는지 없는지 기획에서 불명확

---

## 4. 수익화 구조 불편함

### 4-1. 대화 팩 $0.99/5회는 단위가 너무 작다
- 5회 추가 후 다시 제한에 걸리면 또 결제해야 함
- 자잘한 결제가 반복되는 게 피로감을 줌

### 4-2. 무료 vs 유료 경계가 여러 곳에 있어 복잡하다
- 6스텝 완료 조건, 브로드캐스트 횟수 제한, 1:1 대화 횟수 제한, 스텝 7 페이월
- 각각 어떤 플랜에서 해결되는지 한눈에 파악하기 어려움
- 사용자가 "뭘 사면 이게 풀리지?"를 매번 계산해야 함

### 4-3. 3일 무료 체험 후 유료 전환 여부 판단이 어렵다
- 6스텝 완료에 3일이 걸릴 수도 있음 → 소통 탭을 아직 못 써봤는데 체험 기간 종료
- 핵심 기능(소통)을 체험하기 전에 결제 여부를 결정해야 할 수 있음

## 5. 전반적인 불편함

### 5-1. iOS 전용 출시 (초기)
- Android 사용자는 앱 자체를 쓸 수 없음
- 글로벌 서비스인데 iOS만 지원하면 대부분의 비영어권 사용자가 이탈

### 5-2. 오프라인 모드 없음
- 지하철 등 인터넷이 불안정한 환경에서 학습 불가
- 음성 파일이 미리 로드된 경우에도 AI 피드백이 오프라인에서 안 됨

### 5-3. 학습 진도 관련 유연성 부족
- 한 스텝을 중간에 그만뒀다가 나중에 다시 돌아오는 흐름이 명확히 기획되어 있지 않음
- 여러 언어를 동시에 학습하고 싶을 때 (예: 영어 + 일본어) 지원 여부 불명확

### 5-4. 알림의 구체적인 모습이 없다
- 스트릭 유지 알림이 얼마나 자주 오는지, 끄는 방법이 얼마나 쉬운지 불명확
- 과도한 푸시 알림은 앱 삭제로 이어짐

### 5-5. Ori 캐릭터의 역할이 불명확
- 온보딩, 빈 상태, 피드백 등에 등장한다고 하는데
- "귀여운 캐릭터 있음" 이상의 역할이 없으면 사용자 기억에 남지 않음
- 캐릭터가 학습 동기 부여에 실제로 기여하는지는 미지수
